This is the published documentation site for `spreadsheet-handling`. The main
user entry point remains the
link:{demo-url}[spreadsheet-handling-demo repository]; the link:{project-url}[core repository]
hosts the library, framework documentation source, and release tooling.

== Documentation Entry Points

* link:latest/core/[Latest core documentation] -- moves with the newest released
  core tag.
* link:versions/{project-version}/core/[Core documentation for `{project-version}`]
  -- stable snapshot for the release that generated this index page.

Older releases live under stable paths of the form `versions/<tag>/core/`. They
are intentionally immutable per tag, so links from old PyPI release pages keep
resolving to the documentation that was current at the time of that release.

== Scope

This first publishing slice owns only the core documentation subtree
(`versions/<tag>/core/` and `latest/core/`) plus this root navigation page.
Demo walkthrough output (`demo/`) and presentation output (`presentations/`)
are reserved for later publishing slices.
