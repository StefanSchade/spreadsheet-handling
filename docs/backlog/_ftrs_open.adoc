phases below (it does not renumber them): model and metadata contract first, then
the ODS capability contract, then carrier-neutral hygiene, then ODS internals,
and only then the later visible-structure features. Read this first for "what
comes before what".

* Modeling / inventory slices carry no runtime implementation.
* Adapter stabilization is behavior-preserving and guard-backed.

  prerequisites land.
