===== FTR-DISTRIBUTION-PEX — Self-contained CLI runner for non-developer users

** *Purpose:*  Deliver a single executable (`.pex` / `.pyz`) of spreadsheet-handling for analysts without Python installs.

** *Scope:*  Applies to the maintained CLI tools (`sheets-run`,
`sheets-schema-maintain`).

** *Solution:*
- Add a Make target and CI job building a PEX or shiv archive from the current wheel.
- Embed version info and reproducible hash for traceability.
- Provide a short usage note for analysts (Windows/macOS/Linux).

** *Acceptance:*
- CI produces PEX artifacts for the maintained CLI tools on release tags.
- Analysts can execute it without prior setup; output identical to CLI invocation.
- Docs include 1-page distribution guide.
