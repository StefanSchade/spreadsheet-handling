== Action-major-version finding from Codex

Codex verified the current upstream action majors during review and found that
the repository-local claims are stale:

* `actions/checkout`: workflows use `@v4`; current upstream documentation
  advertises `@v6`.
* `actions/setup-python`: workflows use `@v5`; current upstream documentation
  and releases advertise `@v6`.
* `actions/setup-java`: workflows use `@v4`; current upstream documentation
  and releases advertise `@v5`.
* `pypa/gh-action-pypi-publish@release/v1` remains appropriate for the PyPI
  Trusted Publisher job.

This is not merely wording drift. The FTR acceptance criteria include current
supported GitHub Action major versions and Node.js runtime deprecation
handling. Because current workflow pins lag upstream majors that target the
newer Node runtime, the implementation is *not yet fully accepted* and should
not be pushed from `dev` until the pins are updated or the maintainer records
a conscious decision to defer the bump.

== Residual risks

* *First-run validation on the new majors.* Bumping three actions in one
  cycle is operationally sound but the new majors should be watched on
  the first GitHub-hosted run: checkout credential handling, Python
  version-cache layout, and the Java toolchain cache may have rolled
  between majors. Roll back to the previous major (`@v4`/`@v5`/`@v4`) is
  trivial if anything regresses, since the YAML change is a one-line
  pin per usage.
* *Dependabot 403 handling still requires manual GitHub UI inspection.*
  The code review can prove there is no repo-local `dependabot.yml`,
  but it cannot see or fix repository Security / Code-security
  settings. Re-stated from the original review; unchanged.
* *Repository-level Actions default.* The workflow `permissions:`
  blocks harden current workflows, but the
  `Settings -> Actions -> General -> Workflow permissions` default
  should still be confirmed as the most-restrictive option in the
  GitHub UI so future workflows do not inherit write access by
  omission. Re-stated from the original review; unchanged.
