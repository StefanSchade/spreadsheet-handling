=== Purpose of This Guide

This guide is the primary reference for developers working on `spreadsheet-handling`.
It covers project setup, tooling, conventions, testing, the AI-friendly development model,
and the CI/CD pipeline that publishes releases to PyPI.

The guide is intentionally structured to be useful both to human readers and to AI agents
collaborating on the project — a first-class concern in this codebase.
See <<ch05_ai_friendly_development>> for details.

Canonical machine-readable AI context lives in `docs/ai_info/`; this guide is the
human-facing companion that explains the *why* behind those rules.
