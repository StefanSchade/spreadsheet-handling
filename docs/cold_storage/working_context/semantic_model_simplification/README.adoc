= Semantic Model Simplification Evidence
:revdate: 2026-06

== Purpose

This working-context area collects evidence discovered during Functional Model
promotion work where terms, sections, or review artifacts appear to add
semantic-model complexity without clear functional need.

It is an evidence log only.

It is not:

* a backlog;
* a to-do list;
* a semantic-model redesign;
* an FTR staging area.

== Use

Future Functional Model promotions may append observations here when a review
finds deletion-oriented evidence: coined terms that restate plain intent,
semantic bookkeeping that belongs in a later semantic pass, local concepts that
should stay local, or unresolved terms that need more evidence before any
decision.

Entries should record observations and sources, not prescribe changes to the
Semantic Model. Any later work that changes the Semantic Model should use normal
FTR/backlog mechanisms outside this evidence area.
