== Cold Storage

Archived material that should not be pulled into the normal working context.
Use this area only when a task explicitly requires historical tickets,
project-history archaeology, or migration of retired documentation.

Machine-readable archive indexes live next to the archived material as
`index.json` files. They are intentionally flat arrays so they can later be
consumed by spreadsheet-handling pipelines.

Main archive areas:

* `backlog/` -- closed, superseded, and historical backlog artifacts.
* `concepts/` -- retired concept notes and historical problem-space material.
* `reviews/` -- completed global and topical reviews.
* `collaboration/` -- historical collaboration / repository-roundtrip
  analysis.
* `project_history/` -- older project-history snapshots.
