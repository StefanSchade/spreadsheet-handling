== Registries

Reviewed registry artifacts that describe maintained project contracts.

These registries are documentation-side guardrail artifacts.
They are validated by architecture tests.

The `registry_semantics/` subsection that follows defines the foundation
vocabulary used by the registries — most importantly the meta registry — and
should be read before the registries themselves when the meaning of
`canonical`, `derived`, `carrier`, `model`, or `presentation` is in doubt.
