=== Practices

- Primer prompts defining coding and documentation rules are stored under `docs/internal_guide/primer_prompts/`.
- Each generated artifact is reviewed, version-controlled, and tested.
- AI-assisted work must call out planned changes to outward-facing interfaces before implementing them.
- If a change affects consumers such as `spreadsheet-handling-demo`, YAML schemas, CLI usage, or documented plugin conventions, explicit approval is required first.
