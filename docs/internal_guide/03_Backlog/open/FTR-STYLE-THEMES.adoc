* FTR-STYLE-THEMES — Themes & helper band highlighting

** *Status:* New (consolidates existing behavior)  |  *Epic:* Phase 3 — Functional Parity and IR-driven Features

** *Context*::
Header fill, bold fonts, freeze panes, and AutoFilter improve readability. Portions exist; formalize & centralize via IR StylePass.

** *Goal*::
Expose global/per-sheet style options in meta and apply them in an IR StylePass for spreadsheet outputs.

** *Rationale*::
Centralized styling avoids duplication in adapters and keeps Excel UX consistent. (Existing code demonstrates the approach.)

** *Solution Design*::
- StylePass applies theme tokens (headers, helpers, data); configurable colors and freeze/autofilter.

** *Acceptance*::
- Global/per-sheet options: `auto_filter`, `freeze_header`, `header_fill_rgb`, `helper_fill_rgb`.
- StylePass applies to all sheets; renderer writes final styles.
- Tests confirm presence of AutoFilter and header style in output workbooks.
- Acceptance: visual diff stable; helper columns/rows clearly marked; options in config.
