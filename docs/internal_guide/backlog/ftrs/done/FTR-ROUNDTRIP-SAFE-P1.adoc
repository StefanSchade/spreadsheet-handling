===== FTR-ROUNDTRIP-SAFE-P1 ‚Äî Controlled round-trip for spreadsheet layouts

* *Status* Done
* *Context* Workbooks produced by the IR renderer need a controlled re-import path so that edits (appending rows, changing values) can be round-tripped without structural drift.
* *Goal* Provide `parse_ir` ‚Äî a minimal parser that reconstructs a canonical `WorkbookIR` from XLSX files written by our renderer, covering Scenario A (flat tables with anchors).
* *Rationale* Closes the write‚Üíedit‚Üíre-import loop for the most common layout pattern (one flat table per sheet with optional MultiIndex headers).
* *Acceptance*
- `parse_ir` reconstructs the same logical table set (ids, order, shapes) for Scenario A.
- Embedded `_meta` sheet is read and hidden; explicit anchors override meta.
- MultiIndex headers detected via horizontal/vertical merges.
- Merges, header grids, validations, and freeze panes are extracted.
- 11 round-trip tests cover flat table geometry, multiple sheets, data count, MultiIndex detection, merge extraction, header grid, hidden meta, freeze panes, auto filter, validation extraction, and empty-sheet edge case.

*Delivery scope*

[cols="1,3"]
|===
| Parser | `rendering/parse_ir.py` ‚Äî `parse_ir(path, anchors, stop_on_empty_row, stop_on_empty_col) ‚Üí WorkbookIR`
| Meta discovery | `_read_meta_sheet()` ‚Äî reads hidden `_meta` sheet; JSON + `ast.literal_eval` fallback for Python repr blobs
| Header detection | `_detect_header_rows()` ‚Äî horizontal merge ‚Üí 2 header rows, vertical merge ‚Üí max depth
| Table parsing | `_parse_table_block()` ‚Äî anchor + empty-row/col sentinel; populates `TableBlock` with headers + data
| Merge extraction | `_extract_header_merges()` ‚Äî collect `MergedCell` regions in the header area
| Header grid | `_extract_header_grid()` ‚Äî rebuild per-cell header values for merged regions
| Validations | `_extract_validations()` ‚Äî map openpyxl `DataValidation` to `DataValidationSpec`
| Freeze panes | `_extract_freeze()` ‚Äî read `sheet_view.pane` for freeze row/col
| Tests | `test_parse_ir_roundtrip.py` ‚Äî 11 tests (flat geometry, multi-sheet, MultiIndex, merges, header grid, meta, freeze, autofilter, validations, empty sheet)
|===

*Design Principles*

- Stable anchors over heuristics: prefer explicit coordinates; embedded meta wins over heuristics.
- Controlled edit surface: appending rows, inserting columns, editing values are safe; structural ops that break anchors are out of scope.
- Lossless data, lossy style: guarantee round-trip for data and metadata; styling may be normalized.

*Out of Scope (P1)*

- Scenario B (named ranges) ‚Äî deferred to a later FTR.
- Complex merges in headers beyond what `_detect_header_rows` infers.
- CLI/pipeline flags (`--meta-source`, `--table`, etc.) ‚Äî deferred.
- Arbitrary user-inserted shapes/charts/pivots.
