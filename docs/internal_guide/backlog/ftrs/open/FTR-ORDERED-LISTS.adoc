===== FTR-ORDERED-LISTS -- Preserve order in extracted 1:n child tables

*Purpose:*::
Model JSON lists as separate child tables with `parent_id` and `pos` columns so
that recomposition preserves deterministic list order.

This is a follow-up concern on top of nested-list extraction, not a standalone
answer to the broader bundling/layout problem.

*Depends-on:*::
FTR-NESTED-LISTS
