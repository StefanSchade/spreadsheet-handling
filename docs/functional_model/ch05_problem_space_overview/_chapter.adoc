== Problem-Space Overview

[NOTE]
====
*Scope limit.*

This chapter is a lightweight, explanatory entry path into the project's
problem space. It is *not* an ontology, taxonomy, semantic-set model,
lifecycle classification, registry vocabulary, or rule-ready contract
model. Future classification work belongs in the Semantic Model and is
routed through the Semantic Model's Adoption Strategy slice rules, not
through this chapter.

If a reader notices that this chapter is growing classification material,
treat that as a signal to file a follow-up FTR rather than extend the
overview in place.
====

This chapter exists so that the project's problem-space framing has a
named, navigable home in the Functional Model. It absorbs the framing of
the German archived concept under
`docs/cold_storage/concepts/problemraum_konzept/` (chapters 02 and 03) into
English without replacing it. The German source remains the preserved
working input and is not modified by this chapter.
