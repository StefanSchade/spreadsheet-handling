== Primary Scenarios

Software configuration:

* Feature matrices.
* Product variants.
* Market-specific settings.
* Validation mappings.
* Workflow definitions.
* API mappings.
* Pricing and regulatory parameters.

Controlled spreadsheet-based operations:

* Operational checklists.
* Reference data.
* Governance inventories.
* Coordination workflows.
* Existing IDV-style processes that need reproducibility and tests.

External data curation:

* External APIs expose structured or semi-structured data.
* Humans review, correct, enrich, or classify the data in spreadsheet form.
* The result returns to structured downstream processing.

Operational and DevOps configuration:

* Release matrices.
* Deployment matrices.
* Compatibility inventories.
* Support mappings.
* Environment-specific parameters.
