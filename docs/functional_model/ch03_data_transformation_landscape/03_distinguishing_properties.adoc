== Distinguishing Properties

Spreadsheet projections are first-class:

* The spreadsheet is an editable projection.
* It is not only an import or export format.
* It is not the canonical model.

Roundtrip reconstruction is central:

* Export and import are treated as one semantic cycle.
* Metadata must support reconstruction.
* Human edits must remain mappable to canonical data.

Semantics are separated:

* Canonical domain meaning.
* Projection meaning.
* Rendering concerns.
* Metadata layers.
* Persistence details.
* Transformation steps.

Human editing stays compatible with software discipline:

* Deterministic processing.
* Testability.
* Explicit identifiers.
* Reproducible transformations.
* Git-friendly artifacts.
