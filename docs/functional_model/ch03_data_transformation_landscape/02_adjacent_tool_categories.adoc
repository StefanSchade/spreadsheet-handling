== Adjacent Tool Categories

[cols="1,1,1", options="header"]
|===
|Category
|Usually good at
|Common gap for this project

|Spreadsheets
|Visibility, flexibility, mass editing, dense comparison
|Implicit semantics, weak identifiers, fragile conventions, weak pipeline integration

|YAML / JSON configuration
|Explicit structure, Git workflows, automation, reproducibility
|Hard visual comparison, costly mass editing, poor fit for matrix work

|Low-code / no-code platforms
|GUI assembly, quick workflow setup, accessibility
|Opaque semantics, vendor persistence, weak deterministic reconstruction

|ETL / data engineering frameworks
|Transformation, normalization, repeatable pipelines
|Human spreadsheet editing is usually an edge case

|Collaborative data SaaS
|Shared table editing, lightweight relational concepts, convenience
|Hosted product model, weaker Git and pipeline orientation
|===
