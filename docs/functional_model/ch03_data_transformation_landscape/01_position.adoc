== Position

The project sits between:

* Spreadsheet-centric work.
* Structured data transformation.
* Configuration management.
* Lightweight master data handling.
* AI-assisted structured workflows.

Its center:

* Deterministic spreadsheet projections.
* Canonical structured data.
* Explicit semantics.
* Roundtrip reconstruction after human editing.

Practical gap:

* Spreadsheets are often operationally unavoidable.
* Structured semantics are still required.
* Existing tools often cover only one side well.
