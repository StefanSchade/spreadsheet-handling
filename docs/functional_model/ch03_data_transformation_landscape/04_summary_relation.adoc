== Summary Relation

The project can be described as:

* Spreadsheet-first editing for canonical structured data.
* Explicit semantic modeling for workflows that already use spreadsheets.
* Deterministic reconstruction after human review and editing.

Relation to adjacent tools:

* More structured than ordinary spreadsheet workflows.
* More human-editable than YAML or JSON only workflows.
* More explicit and reproducible than low-code abstractions.
* More spreadsheet-aware than typical ETL systems.
* More pipeline-oriented than collaborative table SaaS.
