== Purpose

This area keeps active working documents.

It is not a stable documentation surface.

Use it for:

* review material still needed for follow-up work,
* interim analysis that has not yet been consolidated,
* concept drafts that may still affect current planning.

Stable destinations:

* agent guidance: `docs/ai_info/`
* backlog: `docs/backlog/`
* developer workflow and release management: `docs/developer_guide/`
* functional model: `docs/functional_model/`
* technical model, registries, guardrails, glossary: `docs/technical_model/`
* retired material and completed review packages: `docs/cold_storage/`

Completed working documents should be moved to cold storage with a local
`index.json` entry instead of remaining here as background noise.
