= Table Contract: releases
:toc:

== Purpose

`releases` records release context relevant to project memory.

This table is informational. It does not replace package metadata, tags, or release notes.

== Fields

[cols="1,1,3"]
|===
| Field | Required | Contract

| `id`
| yes
| Stable release context id.

| `version`
| yes
| Version string.

| `status`
| yes
| Controlled value. See <<statuses>>.

| `package_pin`
| no
| Package pin used by the project-memory workspace, if any.

| `date`
| yes
| Release or context date. Existing rows use timestamp strings.

| `note`
| no
| Human-readable note.
|===

== Statuses [[statuses]]

[cols="1,3"]
|===
| Value | Meaning

| `current`
| Current release context for project-memory data.

| `historic`
| Retained historical release context.

| `planned`
| Planned but not current.
|===

== Non-Goals

* No release notes duplication.
* No replacement for Git tags or package metadata.
